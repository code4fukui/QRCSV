# QRCSV
QRコード上でCSVを表現するフォーマットです。

## デモ
[QRCSV デモページ](https://code4fukui.github.io/qrcsv/)

## 機能
- QRコード上でCSVデータを表現できる
- QRコードからCSVデータを読み取れる

## 使い方
QRコードからのCSVデータの読み取り:
1. QRコードリーダーを使ってQRコードをスキャンすると、CSVデータが表示されます。
2. [stop]ボタンを押すとQRコードの読み取りを停止できます。

QRコードへのCSVデータの書き込み:
1. CSVデータを用意します。
2. `<qr-code>` タグを使ってQRコードを生成できます。
   - CSVデータをそのまま `<qr-code>` タグの中に書きます。
   - または、`src="test.csv"` のようにCSVファイルへのパスを指定できます。
